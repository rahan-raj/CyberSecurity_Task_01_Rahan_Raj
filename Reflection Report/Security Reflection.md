## Security Reflection

### Cybersecurity Risks Identified in My Own Digital Habits

After completing this self-audit, I identified several cybersecurity risks in my own day-to-day digital habits. The most significant issue I found was that my Windows 10 operating system was not fully updated. Running an outdated OS is a serious vulnerability because unpatched security flaws can be exploited by malware or remote attackers without any action from me.

The second risk I identified was that MFA was only enabled on my Google account, while GitHub and Instagram remained protected by a password alone. This is particularly concerning for GitHub, which holds code repositories, and for Instagram, which is commonly targeted in account takeover attacks. If a password for either of these accounts were ever leaked in a data breach, an attacker would have direct access with no secondary barrier.

I also realized I had not previously thought much about social engineering. I now understand that the biggest vulnerability in any security setup is often human behaviour rather than technology — trusting phone calls, email links, and urgent messages without proper verification.

### Security Improvements I Will Implement Immediately

The first change I will make is running Windows Update and ensuring automatic updates are turned on so my system stays patched going forward. Second, I will enable MFA on GitHub and Instagram using an authenticator app. Third, I will check all my major accounts on HaveIBeenPwned to see if any passwords have been exposed in past breaches.

### Why Cybersecurity Is Important in Today's World

In today's world, nearly every aspect of life — banking, communication, work, health records, and personal identity — exists in digital form. A single security failure can lead to financial loss, identity theft, reputational damage, or even national security threats in the case of critical infrastructure. Cybersecurity is no longer just an IT concern; it is a personal responsibility. Understanding threats and building secure habits is the first and most important step every individual can take to protect themselves and those around them.
